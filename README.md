一、工具简介 / Introduction
中文：本工具为免费公开的民航机队数据查询系统，支持国内所有主流航空公司的机型、飞机注册号反向查询、整机队浏览功能。数据来源于公开民航资讯网站，仅供学习与参考使用。
English：This is a free civil aviation fleet query tool. It supports full fleet browsing, aircraft model checking, and aircraft registration number reverse lookup for all major Chinese airlines. All data is collected from public aviation resources for learning and reference only.
二、功能说明 / Features
中文
- 查询航司全部机队：选择对应航空公司，一键查看该航司所有机型及全部飞机注册号。
- 航司+机型精准查询：选定航司与具体机型，筛选对应所有飞机注册号，数据更精准。
- 注册号反向查询：输入飞机注册号，可反向匹配所属航空公司与具体机型。
- 在线数据更新：支持一键重新爬取最新民航机队数据，实时更新资料库。
English
- Full Airline Fleet Query: Select an airline to view all aircraft models and their registration numbers.
- Precise Model Query: Choose airline + aircraft model to get exact registration list.
- Registration Reverse Lookup: Input aircraft registration number to match airline and model.
- Online Data Update: Support one-click data crawling to refresh the latest fleet database.
三、使用教程 / User Guide
1. 数据初始化 / Data Initialization
中文：首次使用需点击【数据更新/爬取】-【开始爬取航司机队数据】，等待爬取完成后即可正常使用所有查询功能。
English: For first-time use, click "Start Crawling Fleet Data" to initialize the database before querying.
2. 航司机队查询 / Full Fleet Search
中文：切换标签页【查询航司全部机队】，下拉选择航空公司，点击查询即可查看该航司全部机型与注册号。
English: Switch to the first tab, select an airline, and click query to view the full fleet.
3. 机型精准查询 / Model Specific Search
中文：切换标签页【航司+机型查注册号】，分别选择航司和机型，可精准查看该机型所有飞机编号。
English: Switch to the second tab, select airline and model to get precise registration numbers.
4. 注册号反向查询 / Registration Reverse Search
中文：切换标签页【注册号反向查询】，输入飞机注册号（支持模糊/完整号码），一键匹配归属航司与机型。
English: Switch to the third tab, enter the aircraft registration number to find the corresponding airline and aircraft model.
四、注意事项 / Notes
- 中文：本工具数据仅用于学习研究，不可用于商业用途。
- 中文：在线爬取数据为临时容器文件，重启应用会重置，建议本地更新数据后上传。
- 中文：Streamlit 免费应用长期无人访问会休眠，首次打开可能需要几秒重启时间。
- English: This tool is for educational purposes only, not for commercial use.
- English: Online crawled data is temporary and will be cleared after app restart.
- English: Free Streamlit apps may sleep after inactivity, requiring a few seconds to restart.
